AISL Skill-Up Program


AISL은 이론과 실무를 겸비한 인재를 배출하기위하여 과정별로 자율지능시스템에 필요한 지능형 임베디드 시스템 및 백엔드/프론트엔드 시스템 연구개발 수행을 위한 전문성을 갖추기 위하여 "Skill-Up Program" 참여를 요구합니다.
[IoT Basic]
- C/C++
- 아두이노/아두이노 스튜디오
- Arduino Project 1: 아두이노를 활용한 센싱-판단-제어
- Arduino Project 2: 아두이노를 활용한 서버 연동 및 웹/앱기반 제어

[Robot Control]
- C, Python, Java
- Rasp-Pi/OpenCR/Turtlebot
- Linux, ROS
- Turtlebot Project 1: 터틀봇을 활용한 자율주행
- Turtlebot Project 2: 터틀봇을 활용한 서버 구축/연동, 웹/앱기반 제어
- Advanced Project: 터틀봇, 데이터처리, Deep Learning, 서버 로직 개발

[Drone Control]
- C, Python, Java
- Rasp-Pi/Pixhawk FC
- Linux, ardupilot
- Drone Project 1: 드론을 활용한 자율주행
- Drone Project 2: 드론을 활용한 서버 구축/연동, 웹/앱기반 제어
- Advanced Project: 드론, 데이터처리, Deep Learning, 서버 로직 개발

[프론트엔드/백엔드 개발]
- Javascript, Python, HTML, CSS
- Node.js(MEAN Stack) or Django

[클라우드 환경]
- AWS, Azure

[머신러닝 및 데이터 분석]
- Machine Learning basic(Deep Learning concept, PyTorch, Tensorflow/Keras 활용)
- Machine Learning advanced(Deep Learning advanced, Reinforcement Learning)
- Python 패키지 활용 (numpy, pandas, matplotlib, scikit-learn)

[개발환경 및 DevOps]
- IDE: VS Code, Pycharm, IntelliJ etc.
- SCM(Source Code Management): Git&Github
- Docker, Kubernetes
